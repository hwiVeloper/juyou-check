# PRD: 전국 주유소 실시간 유가 조회 웹서비스

## 1. 프로젝트 개요

### 1.1 서비스명
**주유췍** — 전국 주유소 실시간 유가 비교 서비스

### 1.2 배경 및 목적
- 유가가 지속적으로 변동되는 상황에서 소비자가 주변 최저가 주유소를 빠르게 찾을 수 있도록 지원
- 오피넷(Opinet, 한국석유공사) Open API를 활용하여 실시간 유가 데이터 제공
- Next.js 단일 프로젝트 내에서 프론트엔드 + 백엔드(API Routes)를 모두 구현

### 1.3 목표 사용자
- 매일 출퇴근 시 차량을 이용하는 직장인
- 장거리 운행이 잦은 운전자 (배달, 영업직 등)
- 가계 연료비 절감을 원하는 일반 소비자

## 3. 오피넷 Open API 정보

### 3.1 API Key
- 오피넷 홈페이지(https://www.opinet.co.kr) 회원가입 후 무료 발급
- 일일 호출 제한: **1,500 call/일** (무료)
- 환경변수: `OPINET_API_KEY`

### 3.2 사용 API 엔드포인트

| API 이름 | 엔드포인트 | 설명 |
|----------|-----------|------|
| 전국 평균가격 | `GET /api/avgAllPrice.do` | 전국 유종별 평균 유가 |
| 시도별 평균가격 | `GET /api/avgSidoPrice.do` | 시도 단위 유종별 평균가 |
| 시군구별 평균가격 | `GET /api/avgSigunPrice.do` | 시군구 단위 유종별 평균가 |
| 최근 5일 평균가격 | `GET /api/oilDt5AvgPrice.do` | 최근 5일간 전국 일별 평균가 |
| 반경 내 주유소 조회 | `GET /api/aroundAll.do` | 좌표 기반 반경 내 주유소 목록 |
| 최저가 주유소 TOP10 | `GET /api/lowTop10.do` | 전국/지역 최저가 주유소 TOP10 |
| 주유소 상세정보 | `GET /api/detailById.do` | 주유소 ID로 상세 정보 조회 |
| 시도 코드 | `GET /api/sido.do` | 시도 코드 목록 |
| 시군구 코드 | `GET /api/sigun.do` | 시군구 코드 목록 |

### 3.3 유종 코드

| 코드 | 유종 |
|------|------|
| `B027` | 휘발유 |
| `B034` | 고급휘발유 |
| `D047` | 경유 |
| `C004` | LPG (부탄) |
| `K015` | 등유 |

### 3.4 좌표계 주의사항
- 오피넷 API의 `aroundAll`은 **GRS80 / KATEC 좌표계** 사용
- 브라우저의 `Geolocation API`는 **WGS84 (위경도)** 반환
- 백엔드에서 좌표 변환 처리 필요 (PROJ 라이브러리 또는 직접 변환 공식 사용)

## 4. 주요 기능 요구사항

### 4.1 현재 위치 기반 주유소 검색 (핵심 기능)

**기능 설명**
- 브라우저 Geolocation API로 사용자 현재 위치(위경도) 획득
- WGS84 → KATEC 좌표 변환 후 오피넷 `aroundAll` API 호출
- 반경 1km / 3km / 5km 선택 가능 (기본: 3km)
- 유종 선택: 휘발유 / 경유 / LPG / 고급휘발유 (기본: 휘발유)
- 결과를 가격 오름차순으로 정렬하여 카드 리스트로 표시
- 카카오맵에 주유소 마커로 표시 (마커 클릭 시 상세 정보 팝업)

**표시 정보 (주유소 카드)**
- 주유소 이름, 브랜드 (SKE, GS칼텍스, 현대오일뱅크, S-OIL, 알뜰주유소 등)
- 거리 (m / km)
- 선택 유종 가격 (원/L)
- 전국 평균 대비 가격 비교 (저렴 / 비쌈 / 평균)
- 주소

**우선순위**: 🔴 P0 (필수)

---

### 4.2 전국 / 지역 유가 현황 대시보드

**기능 설명**
- 전국 평균 유가 (유종별) 표시
- 시도별 평균가 히트맵 또는 랭킹 테이블
- 최근 5일간 전국 평균가 변동 추이 라인 차트
- 지역 드롭다운 선택 시 해당 시도의 시군구별 평균가 표시

**표시 정보**
- 전국 평균: 휘발유, 경유, LPG, 고급휘발유
- 최저가 지역 / 최고가 지역 하이라이트
- 전일 대비 가격 변동 (▲ / ▼)

**우선순위**: 🔴 P0 (필수)

---

### 4.3 지역별 최저가 주유소 TOP 10

**기능 설명**
- 전국 또는 선택 시도의 유종별 최저가 주유소 TOP 10 목록
- 순위, 주유소명, 가격, 주소 표시
- 카카오맵 연동으로 위치 확인 가능

**우선순위**: 🟡 P1 (중요)

---

### 4.4 상세 주유소 정보 페이지

**기능 설명**
- 주유소명, 브랜드, 주소, 전화번호
- 유종별 현재 판매가격 전체 표시
- 부가 서비스 정보 (세차, 경정비, 편의점, 충전시설 등)
- 영업시간
- 카카오맵 지도 임베드 (해당 주유소 위치)
- 카카오맵 길찾기 링크

**우선순위**: 🟡 P1 (중요)

---

### 4.5 유종 및 반경 필터

**기능 설명**
- 유종 탭 (휘발유 / 경유 / LPG / 고급휘발유)
- 검색 반경 슬라이더 또는 버튼 (1km / 3km / 5km)
- 정렬 옵션: 가격순 / 거리순

## 6. Next.js API Routes (백엔드)

```
/api/gas/around           # 반경 내 주유소 (aroundAll 프록시 + 좌표 변환)
/api/gas/avg-all          # 전국 평균가 (avgAllPrice 프록시)
/api/gas/avg-sido         # 시도별 평균가 (avgSidoPrice 프록시)
/api/gas/avg-sigun        # 시군구별 평균가 (avgSigunPrice 프록시)
/api/gas/trend            # 최근 5일 추이 (oilDt5AvgPrice 프록시)
/api/gas/top10            # 최저가 TOP10 (lowTop10 프록시)
/api/gas/station/[id]     # 주유소 상세 (detailById 프록시)
/api/gas/codes/sido       # 시도 코드 목록
/api/gas/codes/sigun      # 시군구 코드 목록
```

> API Routes는 오피넷 API Key를 서버 환경변수로 관리하여 클라이언트에 노출되지 않도록 처리

### 7.2 모바일 핵심 UX 패턴

#### 바텀시트 (Bottom Sheet)
- 지도와 리스트를 동시에 볼 수 있도록 바텀시트 방식 채택
- **최소화**: 지도 70% + 카드 목록 30% 노출
- **반확장**: 지도 40% + 카드 목록 60% (기본 상태)
- **최대화**: 카드 목록 전체 화면 (지도 숨김)
- 드래그 핸들로 높이 조절 가능

#### 하단 탭 내비게이션 (모바일 전용)
- 엄지손가락 접근 영역 고려, 화면 하단 고정
- 탭: 홈(위치검색) / 유가현황 / TOP10
- 아이콘 + 텍스트 레이블 구성
- 현재 페이지 탭 강조 표시

#### 필터 바텀시트
- 필터 버튼 탭 시 하단에서 슬라이드업
- 유종 선택, 반경 선택, 정렬 기준을 한 화면에서 처리
- "적용" / "초기화" 버튼 제공

#### 주유소 카드 (모바일 최적화)
- 최소 높이 72px (터치 타깃 44px 이상 보장)
- 좌측: 브랜드 뱃지 + 주유소명 + 주소
- 우측: 가격(크게) + 거리 + 평균 대비 표시

#### 인터랙션
- 카드 탭 → 해당 주유소 지도 마커 포커스 + 상세 바텀시트
- 지도 마커 탭 → 카드 목록 해당 항목으로 스크롤
- Pull-to-refresh로 데이터 갱신

## 8. 비기능 요구사항

### 8.1 성능
- 메인 페이지 초기 로드: 3초 이내 (LCP 기준)
- API 응답 캐싱: 전국/시도 평균가는 1시간 캐싱 (Next.js `revalidate` 설정)
- 위치 기반 검색 결과: 5초 이내 응답

### 8.2 보안
- 오피넷 API Key는 서버 환경변수(`process.env.OPINET_API_KEY`)로만 관리
- 클라이언트 번들에 API Key 미포함
- API Routes에서 오피넷 호출 → 클라이언트는 자체 `/api/*` 만 호출

### 8.3 접근성
- 시맨틱 HTML 사용
- 색상 대비율 WCAG AA 기준 준수
- 키보드 네비게이션 지원 (데스크탑)
- 터치 타깃 최소 44×44px 보장 (모바일)
- iOS Safari / Android Chrome 호환성 최우선 검증

### 8.4 SEO
- 정적 메타데이터 설정 (title, description, og:image)
- 대시보드 및 TOP10 페이지: SSR 또는 ISR (1시간 revalidate)
- Open Graph / Twitter Card 태그 설정 (카카오톡 공유 대응)

### 8.5 모바일 브라우저 호환성
| 환경 | 지원 여부 |
|------|-----------|
| iOS Safari 16+ | ✅ 필수 |
| Android Chrome 110+ | ✅ 필수 |
| Samsung Internet | ✅ 권장 |
| 데스크탑 Chrome / Edge | ✅ 지원 |
- `position: fixed` 버그 대응 (iOS Safari 가상키보드)
- `Geolocation API` HTTPS 필수 → Vercel 배포 시 자동 대응

## 11. 제약 사항

- 오피넷 무료 API: 일 1,500 call 제한 → 서버 캐싱 필수
- `aroundAll` API 반경 최대 5km 제한
- 오피넷 API는 HTTP 기반 (HTTPS 미지원) → 백엔드에서 호출 필요 (클라이언트 직접 호출 불가)
- 좌표계 변환 필요: 브라우저 WGS84 → 오피넷 KATEC

## 12. 데이터 흐름

```
브라우저 (Geolocation API)
    │ WGS84 좌표 (위도/경도)
    ▼
Next.js API Route (/api/gas/around)
    │ WGS84 → KATEC 좌표 변환
    │ OPINET_API_KEY 첨부
    ▼
오피넷 API (aroundAll.do)
    │ 주유소 목록 + 가격 JSON 반환
    ▼
Next.js API Route
    │ 데이터 가공 (거리 계산, 브랜드 매핑 등)
    ▼
브라우저 → 카드 렌더링 + 카카오맵 마커 표시
```
